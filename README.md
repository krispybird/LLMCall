# IAT 806 Example - Making an API call to a large language model
 Code examples for IAT 806 - making an API call

 This code makes a call to OpenAI's API using the HTTP Request library by Rune Madsen and Daniel Shiffman
 It returns the results of a prompt to generate a poem about cheese

 Note: An API key must be added, as OpenAI's API service requires payment to access per token
 An alternative list of free LLM services is found here: https://github.com/cheahjs/free-llm-api-resources
